Dales Deep Dive To Dangerous Depths


Synopsis

Dales Deep Dive To Dangerous Depths is a 2d adventure mining game created in one week for Brackeys Game Jam. The game contians a proceduraly generated cave systems with differnet ores and cave types that spawn every game day. The goal of the game is to mine all the way down to the bottom in which the player will find the hidden treasure. However to do this the player must mine till they run out of stamina sell their upgrades to be able to mine deeper the next day with the goal of getting to the treaure in the least amount of days.

Key Contributions


This game was made by me and my friend with me taking all the coding and game design elements and them focusing on art however some of the key contributions are as follows

  • Diverse resource system with using scriptable obejects giving every resource differnt textures, breaking times, prices and rarity
  • Proceduraly generated cave system, with every game day generating a new environemnt with differnt sized caves and resources spawning in veins and clumps throughout in their respective biome
  • Mining system allowing the player to mine with the mouse and different resources having different break times and player feedback popups when mining for amounts
  • Artifact and perks system, with artifacts found in chests giving different perks to the player which are upgradeable giving player freedom in their gameplay
