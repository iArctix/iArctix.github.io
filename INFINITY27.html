INFINITY 27 Bootcamp- Samsara


Overview

A 4 week skill bootcamp in which I was tasked with creating a spell for the live project samsara inside of Unreal Engine. I learned to work using agile methodology using Perforce for source control and Hansoft to manage my tasks and work in an iterative workflow.

Key Points

Some of the key skills aquired were as follows

  • Experience working within a real studio, with collaberation with the other fields such as design and art to create our spell.
  • Experience using industry standard software such as perforce for source control and keeping commits in the right format and hansoft for task management
  • Getting hands on using Unreal Engine 5 for the first time and going from basic knowledge to being able to create a spell working in game as seen in the video
  • Explorative QA testing experience in game and shown standards towards reporting a bug and how to go about it
